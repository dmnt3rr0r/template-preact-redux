## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao sao-first my-project
```

### From git

```bash
sao dmnt3rr0r/template-sao-first my-project
```
